## Installation

Each skill is self-contained. Copy to your agent's skills directory:

```bash
# Clone the repo
git clone https://github.com/mubyn/mubyn-ai-skills.git

# Copy skill to your workspace
cp -r mubyn-ai-skills/arabic-content-calendar ~/.openclaw/skills/
```

Or reference directly in your OpenClaw config:

```yaml
skills:
  - path: ./published-skills/arabic-content-calendar
  - path: ./published-skills/ai-lead-gen-sme
  - path: ./published-skills/ai-coo-protocol
  - path: ./published-skills/shopify-ai-analytics
```
